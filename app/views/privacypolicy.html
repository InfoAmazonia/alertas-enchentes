Política de privacidade

O bot para Facebook do Alerta de Enchentes garante que nenhuma informação pessoal será divulgada.

O bot para Facebook do Alerta de Enchentes garante que só são armazenadas informações públicas usuário caso o mesmo solicite receber notificações recorentes. Essas informações só serão utilizadas para identificar o destinatário das mensagens.

O bot para Facebook do Alerta de Enchentes garante que não enviará mensagens sem o consentimento do usuário.
